SSM - Simple State Machine mixin

SSM is a mixin that adds finite-state machine behavior to an instance of a class.

Example usage: 

  class Door
    include SSM

    ssm_initial_state :closed
    ssm_state :opened

    ssm_event :open, :from => [:closed], :to => :opened do
      puts "Just opened door"
    end

    ssm_event :close, :from => [:opened], :to => :closed do
      puts "Closed door"
    end

  end

  door = Door.new
  door.open
  door.is?(:opened) #=> true
  door.close
